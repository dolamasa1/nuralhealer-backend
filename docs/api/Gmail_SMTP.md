# 📧 Email Notifications - Simple Guide
**NeuralHealer Platform**

---

## 🎯 What Are Email Notifications?

Emails that keep you informed about important events on NeuralHealer:

- Welcome emails when you join
- Security alerts
- Updates from healthcare providers
- Important reminders

## 🔄 How It Works

### Simple Flow:
```mermaid
graph LR
    A[Event Happens] --> B[System Checks]
    B --> C{Email Needed?}
    C -->|Yes| D[Send Email]
    C -->|No| E[Skip]
    D --> F[📬 You Receive It]
```

### Timeline Example:

## 🔧 If You Have Problems

```mermaid

    A[No emails?] --> B[Check spam folder]

```

## ⚙️ Control Your Emails

### Settings Path:
```
Login → Settings → Notifications → Email Preferences
```

### Options:
- **All emails** (recommended)
- **Important only** (security & critical)
- **None** (not recommended)
